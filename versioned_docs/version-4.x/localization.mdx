## Introduction

In any application, there is a need to make the application multilingual. Bow implements a simple translation system.

### Configuration

To use the translation system, you first need to look at the configuration located in the `config/trans.php` file. In this file, the default language is French, so `'lang' => 'fr'`.

Your application's translations are stored in the `frontend/lang` folder. These are `php` files that return an `array`, and as you may notice, there is an `fr` folder and an `en` folder. These are the folders for each language, here `fr` for French and `en` for English; and if there were another language, for example Spanish, the folder name would be `es`.

```txt
/frontend
  /lang
    /en
      messages.php
    /fr
      messages.php
```

All language files return an array of keyed strings.

For example:

```php
// frontend/lang/en/messages.php
return [
  'welcome' => 'Welcome to our application'
];
```

### Configuration de la locale

Your application's default language is stored in the `config/trans.php` configuration file. Of course, you can change this value to suit your application's needs. You can also change the active language at runtime using the `setLocale` method on the [`Bow\Translate\Translator`](https://bowphp.com/api/master/Bow/Translate/Translator.html) class:

```php
use Bow\Translate\Translator;

$app->get('docs/:locale', function ($locale) {
  Translator::setLocale($locale);
  //
});
```

You can configure a `lang` that will be used when the active language does not contain a given translation string. Like the default language, the fallback language is also configured in the `config/trans.php` configuration file:

```php
'lang' => 'en',
```

## Récupération de chaînes de traduction

You can extract lines from language files using the `trans` or `t` helper function. The `t` method accepts the file and the key of the translation string as its first argument. For example, let's retrieve the welcome translation string from the `frontend/lang/messages.php` language file:

```php
echo t('messages.welcome');

// Or alternatively
echo trans('messages.welcome');

// Or alternatively
use Bow\Translate\Translator;

echo Translator::translate('messages.welcome');
```

### Détermination de la locale actuelle

You can use the `getLocale` and `isLocale` methods on the `Translator` class or via the `trans` helper to determine the current locale or check whether the locale matches a given value:

```php
use Bow\Translate\Translator;

$locale = Translator::getLocale();

if (Translator::isLocale('en')) {
 //
}
```

Via the `trans` helper

```php
$locale = trans()->getLocale();

if (trans()->isLocale('en')) {
 //
}
```

If the specified translation string does not exist, the `t` function will return the key of the translation string. So, using the example above, the `t` function would return `messages.welcome` if the translation string does not exist.

### Remplacement de paramètres dans les chaînes de traduction

If you wish, you can define placeholders in your translation strings. All placeholders are wrapped in `{}`. For example, you can define a welcome message with a placeholder name:

```php
'welcome' => 'Welcome, {name}',
```

To replace the placeholders when extracting a translation string, pass an array of replacements as the second argument to the `t` or `trans` function:

```php
echo t('messages.welcome', ['name' => 'Galy']);
```

## La pluralisation

Pluralization is a complex problem, as different languages have a variety of complex rules for pluralization. By using a "pipe" character (`|`), you can distinguish the singular and plural forms of a string:

```php
"names" => "C'est un utilisateur|Ce sont des utilisateurs",
```

After defining a translation string with pluralization options, you can use the `t` or `trans` function. In this example, since the number is greater than one, the plural form of the translation string is returned:

```php
echo t("messages.names", count($names) > 1);
```

With the data replaced

```php
"names" => "Bonjour {name}|Bonjour à tous",
```

```php
echo t("messages.names", ["name" => "Newt"], count($names) == 1);
// => Bonjour Mewt
```
