## Introduction

An HTTP request is the message that the user sends to the Bow application through their browser or other tools such as `curl`, `httpie`, etc.

This message is handled by Bow via the `Bow\Http\Request` class. This class provides access to the request information: IP address, HTTP method, form data, uploaded files, etc.

## Accessing the request

### Via the helper

```php
$request = request();
$name = $request->get('name');
```

### Via dependency injection

```php title="routes/app.php"
use Bow\Http\Request;

$app->get('/', function (Request $request) {
  return $request->get('name');
});
```

### Via a controller

```php
namespace App\Controllers;

use App\Models\User;
use Bow\Http\Request;

class UserController
{
  public function show(Request $request, int $id)
  {
    $user = User::retrieve($id);
    $user->name = $request->get('name');
    $user->persist();

    return view('user/profile', ['user' => $user]);
  }
}
```

## Retrieving data

### Retrieving a value

```php
// Via the get() method
$name = $request->get('name');

// Via the magic property
$name = $request->name;

// With a default value
$name = $request->get('name', 'Invité');

// With a default callback
$name = $request->get('name', fn() => User::first()->name);

// input() is an alias of get()
$name = $request->input('name', 'Invité');
```

### Retrieving a boolean

`boolean()` casts an input to a real `bool`. The values `1`, `"1"`, `true`,
`"true"`, `"on"` and `"yes"` are considered `true`; everything else is `false`.
Handy for checkboxes and feature toggles:

```php
// ?subscribe=on  ->  true
if ($request->boolean('subscribe')) {
  // The box is checked
}

// Absent key returns the default (false by default)
$remember = $request->boolean('remember', true);
```

### Retrieving all the data

```php
$data = $request->all();
```

### Data by source: query string vs POST

When you want to explicitly read `$_GET` or `$_POST` (instead of the
merged array returned by `all()`), use `query()` and `post()`:

```php
// Full $_GET array
$queryParams = $request->query();

// A specific $_GET key (returns null if absent)
$page = $request->query('page');

// With a default value when the key is absent
$page = $request->query('page', 1);

// Full $_POST array
$body = $request->post();

// A specific $_POST key (with an optional default)
$name = $request->post('name', 'Anonymous');
```

:::tip When should you prefer `get()`?
`get($key)` first looks in the merged routes/query/body and offers a
default fallback (`get('name', 'Invité')`). Reserve `query()` / `post()`
for cases where the origin of the data matters (e.g. *never* reading
`?admin=1` from the URL).
:::

### JSON and PUT bodies

Bow automatically decodes the request body: if the `Content-Type`
header is `application/json`, the JSON is decoded and merged into the
data accessible via `get()`, `all()` and `post()`. No manual step
is required to consume a JSON API:

```php
// POST request with Content-Type: application/json and body {"name": "Franck"}
$name = $request->get('name'); // "Franck"
$body = $request->post();       // ['name' => 'Franck']
```

`PUT` request bodies (form-urlencoded) are also parsed
automatically and available through the same methods.

:::warning Invalid JSON
If the `Content-Type` is `application/json` but the body is not valid
JSON, Bow throws a `Bow\Http\Exception\BadRequestException`.
:::

### Filtering data

```php
// Retrieve only certain fields
$data = $request->only(['name', 'email']);

// Exclude certain fields
$data = $request->ignore(['password', 'password_confirmation']);

// Both also accept a list of arguments
$data = $request->only('name', 'email');
$data = $request->ignore('password', 'password_confirmation');
```

### Checking existence

```php
if ($request->has('name')) {
  // The field exists
}
```

### Retrieving old values

After a redirect with validation errors:

```php
$oldName = $request->old('name', 'Valeur par défaut');
```

## Uploaded files

### Retrieving a file

```php
$file = $request->file('avatar');

if ($file) {
  $file->moveTo('/uploads/avatars');
}
```

### Multiple files

```php
$files = $request->file('documents');

foreach ($files as $file) {
  $file->moveTo('/uploads/documents/' . $file->getClientFilename());
}
```

### Checking existence

```php
if (Request::hasFile('avatar')) {
  // A file has been uploaded
}
```

## Request information

### URL and path

```php
// Request path (e.g. /users/1)
$path = $request->path();

// Full URL (e.g. https://example.com/users/1)
$url = $request->url();

// Origin (e.g. https://example.com)
$origin = $request->origin();

// Hostname (e.g. example.com)
$host = $request->hostname();

// Domain without port (e.g. example.com)
$domain = $request->domain();
```

### Checking the path

```php
if ($request->is('users/*')) {
  // The request starts with /users/
}
```

### HTTP method

```php
$method = $request->method(); // GET, POST, PUT, DELETE, etc.

// Quick checks
if ($request->isGet()) { }
if ($request->isPost()) { }
if ($request->isPut()) { }
if ($request->isDelete()) { }

// Match an arbitrary method (case-insensitive)
if ($request->isMethod('patch')) { }
```

### Request type

```php
// AJAX request
if ($request->isAjax()) {
  // AJAX handling
}

// Request expecting JSON
if ($request->wantsJson()) {
  return response()->json($data);
}
```

## Client information

### IP address and port

```php
$ip = $request->ip();
$port = $request->port();
```

### User Agent

```php
$userAgent = $request->userAgent();
```

### Language and locale

```php
$lang = $request->lang();     // e.g. "fr"
$locale = $request->locale(); // e.g. "fr_FR"
```

### Referer

```php
$referer = $request->referer();

if ($request->isReferer('https://google.com/*')) {
  // The user comes from Google
}
```

## Protocol and security

```php
$protocol = $request->protocol(); // http or https

if ($request->isSecure()) {
  // HTTPS connection
}

if ($request->isProtocol('https')) {
  // Same thing
}
```

## HTTP headers

### Retrieving a header

```php
$contentType = $request->getHeader('content-type');
$authorization = $request->getHeader('authorization');
```

### Bearer token

`bearerToken()` extracts the token from an `Authorization: Bearer <token>`
header and returns `null` when it is missing or malformed:

```php
// Authorization: Bearer eyJhbGci...
$token = $request->bearerToken(); // "eyJhbGci..."
```

### Retrieving all headers

```php
$headers = $request->getHeaders();
```

### Checking existence

```php
if ($request->hasHeader('x-api-key')) {
  $apiKey = $request->getHeader('x-api-key');
}
```

## Request identifier

Each request has a unique identifier:

```php
$requestId = $request->id();
// or
$requestId = $request->getId();

// Set a custom ID
$request->setId('custom-request-id');
```

## Session and authentication

### Accessing the session

```php
$session = $request->session();
$session->put('key', 'value');
```

### Authenticated user

```php
$user = $request->user();

// With a specific guard
$user = $request->user('api');
```

## Cookies

```php
// Retrieve all cookies
$cookies = $request->cookie();

// Retrieve a specific cookie
$token = $request->cookie('remember_token');
```

## Validation

Validate data directly from the request:

```php
$validation = $request->validate([
  'name' => 'required|string|max:255',
  'email' => 'required|email',
  'password' => 'required|min:8'
]);

if ($validation->fails()) {
  return back()->withErrors($validation->getMessages());
}
```

## Bags (shared data)

*Bags* allow you to store values within a single request (for
example permissions resolved by a middleware, which a controller will then
read) without polluting the inputs.

```php
// Set / read a value
$request->setBag('user_permissions', ['read', 'write']);
$permissions = $request->getBag('user_permissions');

// Read / replace the entire bags array
$all = $request->getBags();
$request->setBags(['user_permissions' => ['read'], 'feature_flags' => ['v2']]);
```

## Request time

```php
$timestamp = $request->time();
```

## Summary table

| Method | Description |
|---------|-------------|
| `get($key, $default)` | Retrieves a value (all sources merged) |
| `input($key, $default)` | Alias of `get()` |
| `boolean($key, $default)` | Retrieves a value cast to `bool` |
| `query($key = null, $default = null)` | Query string parameters (`$_GET`) |
| `post($key = null, $default = null)` | POST body data (`$_POST`) |
| `all()` | All the data |
| `only($keys)` | Filter by keys |
| `ignore($keys)` | Exclude keys |
| `has($key)` | Checks existence |
| `file($key)` | Retrieves a file |
| `path()` | Request path |
| `url()` | Full URL |
| `method()` | HTTP method |
| `isMethod($method)` | Check the HTTP method |
| `ip()` | Client IP address |
| `getHeader($key)` | HTTP header |
| `bearerToken()` | `Authorization: Bearer` token |
| `session()` | Session instance |
| `user($guard)` | Authenticated user |
| `validate($rules)` | Validates the data |
| `setBag($name, $value)` / `getBag($name)` | Data shared within the request |
